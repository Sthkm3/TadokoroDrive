# Tadokoro's Drive (田所さんのバーチャルドライブ)

掛載後即可體驗野獸先輩的惡臭（？）

![掛載後的虛擬硬碟](img/01.png)

## 簡介

這是一個剩餘空間 114 GB, 容量 514 GB 的虛擬硬碟，經過高度壓縮後只有 94.1 MB。

* **總容量：** 514 GB (符合王道徵收容量)
* **可用空間：** 114 GB
* **掛載佔用空間：** 400 GB

## 使用說明
1. 前往 [Releases](../../releases) 下載壓縮包。
2. 解壓縮得到 `Tadokoro.vhd`。
3. 在該虛擬磁碟圖標上右鍵，點擊掛載。

![掛載](img/02.png)

4. 在本機找到該磁碟
